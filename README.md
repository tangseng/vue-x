# @tangseng/vue-x

> 如果我们项目中用到vuex和axios库，vue-x将能帮助项目减少大量的模板代码，即：保持vuex原本的单向数据流的方式的同时简化代码的书写。

``` sh
npm install @tangseng/vue-x
```
